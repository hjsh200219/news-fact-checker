# 판정 분류 체계 (verdict taxonomy)

## 주장 적격성 라벨 (P7 — 무엇을 검증하는가)
검증 대상은 **경험적으로 확인 가능한 사실 주장**뿐이다. 각 후보 주장을 분류:

| 라벨 | 정의 | 처리 |
|------|------|------|
| `empirical` | 관측·기록·데이터로 참/거짓을 가릴 수 있는 사실 주장 | **검증 대상** |
| `opinion` | 가치판단·해석·논평 | `검증 대상 아님(의견)` |
| `prediction` | 미래 사건 예측 | `검증 대상 아님(예측)` — 단, 근거로 제시된 현재 사실은 별도 검증 |
| `normative` | 당위·정책 주장("~해야 한다") | `검증 대상 아님(당위)` |

## 주장 판정 라벨 (P10 — 6개)

| 판정 | 근거 기준 |
|------|-----------|
| **사실** | 붕괴 후 독립 유효 출처 **≥2**가 주장을 supports, 반박 출처 없음 |
| **대체로 사실** | 핵심은 유효 출처 ≥2로 확인, 지엽적 수치·표현에 경미한 불일치 |
| **일부 사실** | 일부 요소는 확인되나 일부는 미확인 또는 과장/맥락 누락 |
| **오해 소지** | 개별 사실은 맞으나 배치·프레이밍이 오도 (technically true but misleading) |
| **거짓** | 유효 출처 ≥2가 주장을 refutes, 또는 1차 자료와 직접 모순 |
| **검증 불가** | 유효 출처 < 2, 출처 상충, 접근불가, 또는 예산 소진 |

**철칙(H3):** 유효 출처 2개(붕괴 후) 미만이면 절대 `사실`/`거짓`으로 확정하지 않는다 → `검증 불가`.

## 신뢰도 (상 / 중 / 하)
- **상**: 유효 출처 ≥3(그중 1차 자료/공공데이터 ≥1), stance 일치, 취득 경로 clean(STRONG/WEAK_OK).
- **중**: 유효 출처 2, 또는 SUSPECT_OK로 본문 취득, 또는 일부 출처가 재발행.
- **하**: 유효 출처 경계선, 상충 존재, 축소 모드, paywall로 확증 제한.

## 종합 판정 (P11)
주장별 판정을 종합. 핵심 주장 중 `거짓`이 있으면 종합은 최소 `일부 사실` 이하로, `거짓` 다수면 `거짓`.
모두 `사실`/`대체로 사실`이면 종합 `사실`/`대체로 사실`. 상당수가 `검증 불가`면 종합도 `검증 불가` +
낮은 신뢰도, 그 이유(도달성/출처 부족/예산)를 명시.

`[공공데이터]`(정부·공식 통계) · `[모델지식]`(모델 내부지식, 웹 미확인 — 신뢰도 하향 요인).
